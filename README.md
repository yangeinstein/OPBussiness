这个是一人公司信息收集的第一个版本
项目代号： OpenClaw (MoneyMaker)
项目愿景： 为“一人公司”及独立开发者打造 24/7 不间断运行的商业情报大脑。
核心技术栈： Python, Playwright, Serper API, DeepSeek API, Streamlit, Notion API

构建一个能够自动在公网搜集“搞钱思路、SaaS产品案例、独立开发变现”等特定领域文章，并通过大模型提炼核心价值，最终推送到个人知识库（Notion）的智能体。

**⚠️ 初始痛点与挑战：**

1. 传统爬虫无脑抓取，垃圾营销号水文太多。
2. 爬虫缺乏“记忆”，每次运行都在做重复劳动。
3. 纯文本(`.md`)阅读体验极差，无法直观评估情报质量。

4. **核心架构：解耦与防封印设计**
为了打造真正的自治智能体（Autonomous Agent），系统被拆解为高度解耦的五大模块：

1. **进程守护模块 (`doctor.py`)**：负责 24 小时监控主程序，崩溃即重启。
2. **调度大脑 (`autopilot.py` & `brain.py`)**：负责任务派发、大模型调用、预算控制。
3. **感知执行模块 (`search.py` & `browser.py`)**：负责突破防线抓取网页。
4. **可视化前端 (`data_board.py`)**：负责将冰冷的文本转化为可视化数据看板。
5. **异步同步管道 (`sync_to_notion.py`)**：解耦本地抓取与云端同步的物理隔离。

**💡 关键设计决策：**

- 引入 `budget_log.json` 作为“财务总监”，每日 50 次 API 调用强制熔断，保护钱包。
- 引入 `processed_urls.log` 作为“海马体”，实现物理级永久去重。
